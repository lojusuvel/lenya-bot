# Лёня Бот v1.9.3

Это не очередной «полезный ассистент» для погоды. Это полноценный участник чата с характером, памятью и собственным мнением. Лёня — уставший, ироничный, но справедливый. Отвечает коротко, по делу, иногда с матом, если заслужили. Пишет с маленькой буквы, без смайлов (ну почти).

Бот построен на **гибридной архитектуре**. Умеет работать через **OpenRouter** (или любой OpenAI-compatible API), искать информацию в интернете через **Tavily**, а при проблемах автоматически переключаться на бесплатные ключи **Google Gemini**.

## Основные фишки

### Красивые сообщения (Rich Messages)
С обновлением Telegram (Bot API 10.1) Лёня шлёт **красиво оформленные** сообщения: заголовки, списки, таблицы, цитаты, сворачиваемые блоки и спойлеры. Короткие реплики остаются живым текстом, а длинные ответы и отчёты оформляются нормально. На старых клиентах работает авто-фоллбэк в обычный текст.

### Всеядность (Vendor Agnostic)
Лёня не привязан к одной нейросети. В конфиге можно указать любой `BASE_URL`. По умолчанию настроен на **OpenRouter**.

### Поиск в интернете
Лёня не галлюцинирует. Если спросить про новости или курсы — полезет в интернет. Поддержка **Tavily** (1000 бесплатных запросов/мес) или нативный поиск Google как резерв.

### Экономия токенов
Бот разделяет задачи между моделями:
1.  **Logic Model:** `google/gemma-3-27b-it` — анализирует контекст, решает, отвечать ли, ставит реакции.
2.  **Smart Model:** `google/gemini-3-flash-preview` — генерирует ответы.
3.  **Search Model:** Tavily или Perplexity.

### Отказоустойчивость
Если API упал — Лёня не выключится. Автоматически переключится на **Google Gemini**.

### Контекст и память
*   **Время:** Бот знает, что сервер в Токио (UTC+9). Если спросить время — скажет с учётом твоего часового пояса.
*   **Репутация:** Лёня ведёт досье на каждого участника (0-100 баллов). Копить доверие сложно (+1..+3), терять легко (-5..-10). Обижается только если грубят лично ему.

### Голосовые и картинки
*   **Голосовые:** Расшифровывает бесплатно через Google API.
*   **Картинки:** Понимает, что на них изображено.

### Telegram Business
Лёня может работать как ассистент в личных переписках Telegram Business. Отвечает только владельцу business-аккаунта и только по вызову.

## Установка и запуск

Требуется **Node.js 18+**.

### 1. Клонирование
```bash
git clone https://github.com/lojusuvel/lenya-bot.git
cd lenya-bot
npm install
```

2. Настройка (.env)

Создай файл .env в корне проекта.

Минимальный конфиг:

```ini
TELEGRAM_BOT_TOKEN=твой_токен_от_botfather
ADMIN_USER_ID=твой_id_телеграм
```

Полный список настроек — в файле .env.example.

3. Запуск

Локально:

```bash
node src/index.js
```

На сервере (через PM2):

```bash
npm install -g pm2
pm2 start src/index.js --name "lenya-bot"
pm2 save
```

Команды

Для всех

· Текст: Просто общайся. Лёня ответит, если его позвать по имени или ответить на его сообщение.
· лёня напомни [текст] — поставит напоминание.
· лёня кто я? — расскажет твоё досье.
· лёня монетку — орёл/решка.
· лёня число 1-100 — рандомное число.
· лёня кто из нас [вопрос] — выберет случайного участника.
· /start — справка.

Для админа

· /ban @username — забанить пользователя.
· /ban — показать список последних активных (чтобы забанить без юзернейма).
· /unban ID — разбанить.
· лёня стата — статистика токенов.
· /mute — режим тишины в чате.
· /reset — сбросить память бота.

В Telegram Business

· лёня [вопрос] — вызвать Лёню в личной business-переписке.
